# Réplica interactiva del portfolio

Proyecto estático que replica el diseño interactivo del portfolio proporcionado.

Archivos importantes:

- `index.html` — estructura del sitio (hero, about, projects, certifications, contact)
- `style.css` — estilos responsivos
- `script.js` — interactividad (menú, filtros, formulario demo)

Probar localmente:

1. Abrir `index.html` en el navegador (doble clic o arrastrar al navegador).

Desplegar en GitHub Pages (opción sencilla):

1. Crear un repo en GitHub.
2. Desde la carpeta del proyecto ejecutar:

```bash
git init
git add .
git commit -m "Replica interactiva"
git branch -M main
git remote add origin <URL-DE-TU-REPO>
git push -u origin main
```

3. En GitHub: Settings → Pages → Source: `main` branch, carpeta: `/ (root)` → Guardar.
4. Esperar unos minutos; la URL se mostrará en la sección Pages.

Notas:

- El formulario de contacto es demo (no envía emails). Para producción puedes usar Formspree, Netlify Forms o un endpoint propio.
- Si prefieres una réplica con React/Tailwind (más parecida al original), puedo scaffoldearla y añadir instrucciones de build/deploy.
